# split-demo

[Live demo](https://sebastiendavid.github.io/split-demo/build/index.html)

> Built with `node@6.10` & `npm@4.2`.  
> Tested with `chrome@57`, `firefox@52` & `safari@10.1`

## Install

```
npm i
```

## Build

```
npm run build
```

## Dev mode

```
npm start
```

## Prod mode

```
npm run prod
```

## Selenium tests

Start the app server `npm start` or `npm run prod`.  
Then launch tests:

```
npm run wdio
```
